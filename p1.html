1. detailsGenerator This function accepts 2 parameters name and age as a string. You need to return it as an object. Example name="ashish" and age=25 should return { name: "ashish", age: 25 }

2. arrayReverse Here you will have an array of numbers in params. you will need to reverse that array. for example [1,2,3] should be [3,2,1].

3. arraySort Here you will have array of objects in params. You will need to sort the record in accending order by name. Example: javascript const record = [ { id: 1, name: 'robert' }, { id: 2, name: 'downy' }, { id: 3, name: 'junior' }, { id: 4, name: 'captain' }, ] // This record should be sorted as [ { id: 4, name: 'captain' }, { id: 2, name: 'downy' }, { id: 3, name: 'junior' }, { id: 1, name: 'robert' }, ]

Out put in console

4. amendRecord This particular function is to ammend a new key inside existing array of objects. The first parameter is array of object and second parameter is object with name `points`: Example javascript const firstParam = [ { id: 1, name: 'robert' }, { id: 2, name: 'downy' }, { id: 3, name: 'junior' }, { id: 4, name: 'captain' }, ] const secondParam = { robert: 5, downy: 3, junior: 1, captain: 4 } // The final output should be: [ { id: 1, name: 'robert', points: 5 }, { id: 2, name: 'downy', points: 3 }, { id: 3, name: 'junior', points: 1 }, { id: 4, name: 'captain', points: 4 }, ]

5. filterByMaxPoints This function is based on previous function `amendRecord`. once you ammend the record, you need to call this function to get the object which has max points. Example: javascript // From this record [ { id: 1, name: 'robert', points: 5 }, { id: 2, name: 'downy', points: 3 }, { id: 3, name: 'junior', points: 1 }, { id: 4, name: 'captain', points: 4 }, ] // It should return this object { id: 1, name: 'robert', points: 5 }

6. removeDuplicates Here you will have one parameter which is array of objects. You need to remove the record from the array which has duplicate name. Example: javascript [ { id: 1, name: 'robert', points: 5 }, { id: 2, name: 'downy', points: 3 }, { id: 3, name: 'junior', points: 1 }, { id: 4, name: 'captain', points: 4 }, { id: 5, name: 'robert', points: 30 }, ] // This record has 2 records with name robert. So 1 record from these 2 should be deleted and the output should be: [ { id: 1, name: 'robert', points: 5 }, { id: 2, name: 'downy', points: 3 }, { id: 3, name: 'junior', points: 1 }, { id: 4, name: 'captain', points: 4 }, ]

Out put in console
